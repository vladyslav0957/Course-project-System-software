*Опис програми
Програма призначена для очистки комп'ютера від зайвого сміття. Вона буде сканувати вибрані папки, знаходити тимчасові або непотрібні файли і видаляти їх, щоб звільнити місце на диску. Користувач зможе переглянути список знайдених файлів перед тим, як їх стерти.

*Основні можливості (Фічі)
1. Вибір конкретної папки або диска для сканування.
2. Пошук файлів за популярними розширеннями сміття (наприклад .tmp, .log, .bak).
3. Виведення списку всіх знайдених файлів у вікні.
4. Відображення розміру файлів, щоб бачити, скільки місця можна звільнити.
5. Можливість зняти галочку з файлів, які не треба чіпати.
6. Кнопка для повного видалення вибраного сміття.
7. Захист від видалення важливих системних файлів Windows.
8. Збереження звіту (логу) про те, що і коли було видалено.
9. Простий і зрозумілий інтерфейс.
